# 前言

Hexo 博客框架基于 NodeJS 构建，可以快速高效地解析 Markdown 文本，生成网站的各种静态网页，还能通过主题和插件等定制化网站的样式和功能，结合 Github Pages 等服务进行线上部署也很方便。

# 环境准备

首先安装好 NodeJS 和 Git。NodeJS 是必需的运行环境，Hexo 框架及其功能插件大都使用 npm 命令安装；下载主题、线上部署及备份等代码文件则需要 git 命令操作。下载安装工具后，打开 CMD命令控制台 中输入版本查询命令，可以确认各个工具安装是否正常：

# 本地启动
