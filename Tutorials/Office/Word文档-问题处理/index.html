# Word 文档 - 问题处理

# 删除空白页

出现空白页的原因有很多，有些是多余的 【空行】 造成的， 【Backspace】 或 【Delete】 键删除不必要的空行，再进行简单排版即可。也可以用 【shift】 + 【鼠标左键】 选中整页内容将其删除。这是最简单的情况。

  • 分节符造成
    有时候使用 【分节符】 也会产生空白页，此时无法直接删除空白页，需要先删除分节符：
    在 【Office Word】 中，将视图切换为 【草稿】 后，可以看到 【分节符】 所产生的虚线，使用 【Ctrl】 + 【鼠标左键】 选中该行虚线，即可选中分节符将其删除。
    【WPS文字】 不能切换视图为草稿，此时可以通过 【查找替换】 功能 ( 【Ctrl】 + 【F/H】 )，可以找到文本中的分节符，将其替换 【空格】 后，进而删除空白页。这个方法也可用于两个软件中多个空白页的删除。

# 英文直引号变成中文弯引号问题

这是文本校对设置导致的问题。
WPS 文字中，在 【文件】 -> 【选项】 -> 【编辑】 -> 【自动更正】 即可找到 【直引号替换为弯引号】 设置，取消勾选并确认设置即可。WPS 文字默认不勾选该项。
而 Office Word 中，在 【文件】 -> 【更多】 -> 【选项】 -> 【校对】 -> 【自动更正选项】 -> 【键入时自动套用格式】 找到该选项并取消勾选，之后都点击确认即可完成设置。Office 默认勾选该项。

# 取消自动编号

新使用的 Office Word 或者 WPS 文字往往开启了自动编号功能，在书写上一项后，按下 【Enter】 回车即可生成下一项的序号。虽然自动编号有时候能方便书写，但自动编号的格式经常会引起排版不协调的问题，因此，建议关闭这个功能，手动编号。
Office Word 中，在 【文件】 -> 【更多】 -> 【选项】 -> 【校对】 -> 【自动更正选项】 -> 【键入时自动套用格式】 找到 【自动编号列表】 设置，取消勾选即可。
WPS 文字中，在在 【文件】 -> 【选项】 -> 【编辑】 -> 【自动更正】 即可找到 【键入时自动应用自动编号列表】 设置，取消勾选。

# 公式显示问题

因为公式的行高一般并不与文字相同，尤其是使用了多元数学符号的公式，那么在全文设置 【行距】 为 【固定值】 时，公式就有可能无法正常显示，此时，选中公式这一行，单独将其 【行距】 修改为 【单倍行距】 , 公式行就会以自适应的行高进行显示。

# 图片变形或隐藏问题

图片显示问题通常是由于图片的文字环绕方式设置与 【固定值】 行距的设置冲突。
一般来说，图片插入后的默认文字环绕方式为嵌入式，图片占据一行进行显示，若此时行距设置为固定值，图片也只能在一行的固定高度下进行显示，所以就造成了图片变形，或者被隐藏而无法显示的问题。
通常将图片所占行的行距设置为单倍行距后即可正常缩放显示。
将图片的文字环绕方式修改为其他的类型，再通过文字的排版也可以让图片正常显示，但是该方法很可能会让文件由于文字显示区域大小的不同，导致在不同的显示器上，图片显示位置发生偏移或者遮盖住文字的情况。
