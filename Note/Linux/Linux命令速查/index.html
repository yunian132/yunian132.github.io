# crontab 定时模块

前注：
1、远程连接时，若误使用 crontab 命令等其他导致异常的情况，需要使用 'Ctrl + C' 命令先退出异常；
如果直接断开远程连接，将会导致计划列表被清空。
2、定时命令格式为 '* * * * * sh [脚本文件路径 (含文件名及后缀)] [参数 1] [参数 2] ...';
其中 '*' 的位置分别对应分、时、天、月、年的时刻。
